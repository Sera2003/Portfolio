WolFitness – MERN E-commerce Platform

A full-featured MERN-based online fitness store offering user authentication, secure Stripe payments, product browsing, cart management, and a complete admin dashboard for product and order management.

Project Information

  • Category: Web Development
  • Tech Stack: MERN (MongoDB, Express, React, Node.js)
  • Payment Integration: Stripe
  • Role: Full-Stack Developer

Main Features

  • User authentication (JWT-based)
  • Product browsing, filtering, and sorting
  • Shopping cart with quantity and price updates
  • Secure checkout with Stripe or Cash on Delivery
  • Order tracking for users
  • Admin dashboard:
    • Product management
    • Order management
    • User management
    • Sales analytics

Backend Structure

RESTful API with Express/Node.js including routes for authentication, products, orders, and Stripe integration. Admin-protected endpoints handle management tasks.

Frontend Structure

React.js application using Context API. Includes pages for collections, product details, cart, checkout, login/register, and user dashboard.
