React Native E-commerce Mobile App

A cross-platform e-commerce mobile application built with React Native CLI and a TypeScript backend. The app supports Firebase authentication, a MongoDB database, Google Maps address selection, and a full checkout flow with discounts and cart management.

Project Information

  • Category: Mobile Development
  • Frontend: React Native (CLI)
  • Backend: TypeScript, Node.js, MongoDB
  • Auth: Firebase Authentication
  • Maps: Google Maps (lat/long addresses)
  • Role: Mobile & Backend Developer (team project with Git)

Main Features

  • Login / Signup using Firebase authentication.
  • Secure token validation on the backend (Firebase ID tokens verified in TypeScript API).
  • MongoDB database for users, products, carts, orders and promo codes.
  • Display categories and items with filters and sorting (price, category, newest, etc.).
  • Add items to cart with price, description and quantity updates.
  • Checkout flow including address selection via Google Maps and coordinates (latitude/longitude) stored on backend.
  • Promo codes and discounts applied during checkout.
  • Collaborative development workflow using Git branches (feature branches, pull requests, merging).

Backend Structure

The backend is built in TypeScript with Node.js and Express. It exposes REST APIs for authentication, products, cart, orders, addresses and promo codes. Firebase tokens are validated on each protected route. MongoDB stores user profiles, product catalog, cart items, and order history, along with coordinates from Google Maps for delivery addresses.

Frontend Structure

The mobile app is implemented with React Native CLI. It includes screens for authentication, category and product listing, product details, cart, address selection on a map, and checkout. State is managed using React hooks and context; API calls communicate with the TypeScript backend and keep the UI in sync with the cart and order status.
